# STORY-009 - Inspect Artifacts from the Terminal

## User Story

As a developer,

I want to inspect artifacts directly from the command line,

So that I can understand project knowledge without manually locating files.

## Acceptance Criteria

- [ ] I can display a single artifact by ID.
- [ ] I can display multiple artifacts in one command.
- [ ] Artifact types are inferred from their IDs.
- [ ] Original Markdown is preserved.
