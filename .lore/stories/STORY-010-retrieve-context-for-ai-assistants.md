# STORY-010 - Retrieve Context for AI Assistants

## User Story

As a developer using AI assistants,

I want to retrieve artifacts and their relationships,

So that I can provide only the context needed for the current task.

## Acceptance Criteria

- [ ] Direct relations can be displayed.
- [ ] Context can be recursively expanded.
- [ ] Circular references are avoided.
- [ ] Missing artifacts are clearly reported.
- [ ] Context remains deterministic.
