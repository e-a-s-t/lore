# REQ-021 - Show Artifacts by ID

## Requirement

Lore SHALL allow artifacts to be retrieved by their IDs.

## Rationale

Developers and AI agents should be able to inspect artifacts without manually locating files.

## Acceptance Criteria

* [ ] A single artifact can be shown by its ID.
* [ ] Artifact type is inferred from the ID prefix.
* [ ] Requirements, Stories, ADRs, Tests and Features are supported.
* [ ] The command works without requiring knowledge of the underlying directory structure.
