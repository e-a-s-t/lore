# REQ-023 - Preserve Original Markdown

## Requirement

Lore SHALL preserve the original Markdown content when displaying artifacts.

## Rationale

Artifacts should remain human-readable, AI-friendly and suitable for reuse.

## Acceptance Criteria

* [ ] Artifact content is shown without modification.
* [ ] Frontmatter is preserved.
* [ ] Markdown formatting is preserved.
* [ ] Output can be redirected to files or used as input to other tools.
