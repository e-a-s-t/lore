# REQ - AI Friendly

## Requirement

Lore SHALL optimize stored context for AI agents.

## Rationale

AI agents need compact, explicit, linked context rather than scattered documents.

## Acceptance Criteria

- [ ] Artifacts are small and focused.
- [ ] Artifacts have YAML frontmatter.
- [ ] Artifacts contain stable IDs.
- [ ] Artifacts contain explicit relationships.
