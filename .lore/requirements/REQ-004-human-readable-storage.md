# REQ - Human Readable Storage

## Requirement

Lore SHALL store all primary artifacts in Markdown with YAML frontmatter.

## Rationale

Humans and AI agents should both be able to read the repository context directly.

## Acceptance Criteria

- [ ] Artifacts are editable with any text editor.
- [ ] No binary format is required.
- [ ] No proprietary storage format is required.
