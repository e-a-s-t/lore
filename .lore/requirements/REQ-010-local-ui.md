# REQ - Local UI

## Requirement

Lore SHALL provide a lightweight browser UI for browsing artifacts and traceability.

## Rationale

Some relationships are easier to understand visually than in terminal output.

## Acceptance Criteria

- [ ] `lore ui` starts a local web server.
- [ ] The UI requires no external database.
- [ ] The UI shows requirements, ADRs, stories and tests.
