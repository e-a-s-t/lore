# REQ-025 - Support Recursive Context Expansion

## Requirement

Lore SHALL support recursively expanding artifact relationships.

## Rationale

Humans and AI agents should be able to retrieve the complete context surrounding an artifact.

## Acceptance Criteria

- [ ] Related artifacts can be recursively expanded.
- [ ] Circular references are detected and avoided.
- [ ] Artifacts are only shown once.
- [ ] Output remains deterministic.
