# REQ-019 - Prefer Repository Templates Over Built-in Templates

## Requirement

Lore SHALL prefer repository templates when available.

## Rationale

Projects should be able to adapt artifact structures to their own workflows.

## Acceptance Criteria

* [ ] Repository templates override built-in templates.
* [ ] Different artifact types may have different repository templates.
* [ ] Existing behavior remains unchanged when no repository template exists.
