# REQ - Search

## Requirement

Lore SHALL provide full-text search across artifacts.

## Rationale

Developers need to find relevant context quickly from the terminal.

## Acceptance Criteria

- [ ] `lore search audit` returns matching artifacts.
- [ ] Search includes frontmatter and Markdown body.
- [ ] Search works offline.
