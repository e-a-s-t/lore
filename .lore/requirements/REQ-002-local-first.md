# REQ - Local First

## Requirement

Lore SHALL function completely offline.

## Rationale

Knowledge should not depend on SaaS services or network connectivity.

## Acceptance Criteria

- [ ] The CLI works without network connectivity.
- [ ] The UI works without Internet access.
- [ ] No cloud dependency is required for normal use.
