# REQ-016 - Support Feature Artifacts

## Requirement

Lore SHALL support a Feature artifact type.

## Rationale

Features provide a mechanism for grouping related requirements, stories, ADRs and tests into larger business capabilities.

## Acceptance Criteria

- Features can be created.
- Features are stored in .lore/features.
- Features support Markdown content.
- Features participate in traceability analysis.
