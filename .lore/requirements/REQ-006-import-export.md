# REQ - Import and Export Existing Requirements

## Requirement

Lore SHALL import and export artifacts from simple external formats.

## Rationale

Teams often start with requirements in CSV, JSON or exported issue systems.

## Acceptance Criteria

- [ ] CSV import is supported for requirements.
- [ ] JSON export is supported for all artifact types.
- [ ] Imported artifacts receive stable IDs and frontmatter.
